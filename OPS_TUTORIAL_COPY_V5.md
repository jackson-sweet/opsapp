## QUICK REFERENCE: WHAT CHANGED

### Tooltip Changes
- Shorter, more direct commands
- "LONG PRESS" → "PRESS AND HOLD" (clearer for all users)
- "PINCH TO EXPAND" → "PINCH OUTWARD TO EXPAND" (clearer gesture)
- Removed "EXCELLENT!" (OPS voice is confident, not cheerleader)
- Added quotes around button names ("TAP 'CREATE'") for clarity

### Description Changes
- Every step now has purposeful description
- Sample data called out explicitly (clients, crew members)
- Free-form inputs explained ("Type anything")
- Outcomes explained ("Your crew lead will see it")

### Voice Alignment
- Removed corporate/training language ("INITIATED", "TRAINING")
- More direct, practical tone throughout
- Explains WHY things matter, not just WHAT to do
