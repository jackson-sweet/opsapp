# OPS Release Notes

## Version 1.1.0 - Enhanced Contact Management & Improved Scheduling
*Release Date: January 2025*

### What's New

**Advanced Contact Management**
• Manage multiple contacts per client with our new sub-contact system
• Add project managers, site supervisors, and other key contacts to each client
• Quickly access phone and email for any contact with one-tap actions
• Import and export contacts directly to your device's address book

**Smarter Project Scheduling**
• Fixed calendar display to accurately show work days (completion date now correctly represents day after last work day)
• Projects with unscheduled dates now clearly show "Unscheduled" status
• Improved multi-day project visualization across calendar views

**Enhanced Search & Navigation**
• New project search with smart filtering by status, client, and date
• Auto-focusing search fields for faster access
• Role-based project visibility ensures field crews only see their assigned work

**Improved Data Sync**
• Refresh projects on-demand with visual feedback
• Better offline handling with automatic sync when connection returns
• Profile photos now sync for all team members and clients

**UI Refinements**
• Consistent card styling throughout the app with proper borders and spacing
• Address autocomplete powered by MapKit for faster data entry
• Fixed keyboard behavior in search and input screens
• Profile settings now clearly show your email (read-only) alongside editable fields

### Bug Fixes
• Resolved contact sharing issues
• Fixed team member display in project details
• Corrected phone number formatting throughout the app
• Improved preview stability and performance

### Built for the Field
Every update is tested in real job site conditions. We've focused on making OPS more reliable and easier to use with gloves, in bright sunlight, and with limited connectivity.

Your feedback drives our development. Keep it coming.
